# ⚔️ 日语勇士养成 🗡️ - 游戏指南

一个集学习和游戏于一体的交互式日语学习平台。通过完成日语题目来提升勇士的等级和属性！

## 🎮 核心游戏特性

### 1. 🌟 难度系统
游戏开始时需要选择难度：
- **简单** 🌱：适合新手，敌人较弱
- **普通** ⚡：平衡的挑战难度  
- **困难** 👹：强大敌人，获得更多奖励

### 2. 🏆 成就系统
解锁6个成就：
- 初试锋芒 - 完成第一场战斗
- 崭露头角 - 达到5级
- 锋芒毕露 - 达到10级
- 小有资产 - 累计获得500金币
- 完美战士 - 一场战斗中不失血
- 幸运儿 - 获得5个宝箱奖励

### 3. 🎁 宝箱系统
答对题目时有30%的概率获得随机宝箱奖励：
- 💰 金币加成（50+等级*10）
- ⚡ 攻击力提升（+3）
- 🛡️ 防御力提升（+2）
- ❤️ 生命值恢复（+30）
- ⭐ 经验加成（+50）

### 4. 👹 多样敌人
根据难度不同，会遇到不同的敌人：

**简单难度：**
- 史莱姆 (HP:30, 攻击:5, 防御:0)
- 小蛙 (HP:35, 攻击:6, 防御:1)

**普通难度：**
- 妖魔 (HP:50, 攻击:12, 防御:3)
- 骷髅骑士 (HP:60, 攻击:14, 防御:4)

**困难难度：**
- 恶魔统领 (HP:100, 攻击:20, 防御:6)
- 暗黑骑士 (HP:90, 攻击:18, 防御:7)

### 5. 💪 属性升级
通过回答题目升级：
- 每升级一次获得金币和经验
- 每升级一次提升攻击力（+3）、防御力（+2）、生命值（+20）
- 到达特定等级自动升级装备

### 6. 💎 装备系统
- Wooden Sword (起始装备)
- Iron Sword (5级解锁)
- Crystal Sword (10级解锁)
- Legendary Sword (15级解锁)

## 🎯 游戏流程

1. **选择难度** - 在开始界面选择简单/普通/困难
2. **进入战斗** - 点击"进入战斗"按钮
3. **回答题目** - 10道日语题目
   - ✅ **答对**：
     - 造成伤害（攻击力+随机0-4）
     - 获得经验和金币
     - 有30%概率获得宝箱奖励
     - 敌人血量下降
   - ❌ **答错**：
     - 受到敌人伤害（敌人攻击力-防御力）
     - 生命值下降
     - 勇士血量≤0游戏失败

4. **战斗结束** - 击败敌人或勇士被击败
5. **查看成绩** - 查看统计数据和解锁的成就
6. **重新开始** - 点击"重新开始"开启新的战斗

## 📊 游戏界面说明

### 顶部勇士面板
- **等级**：当前勇士等级
- **金币**：累计获得的金币
- **难度**：当前游戏难度标志
- **经验条**：升级进度
- **生命值条**：当前血量比例
- **属性**：显示攻击力和防御力
- **成就徽章**：已解锁的成就显示为金色

### 战斗场景
显示勇士和敌人的对比：
- 角色名称和等级
- 生命值条和具体数值
- 实时更新战斗状态

### 题目区域
- 日语字符展示
- 罗马字标注
- 四个选项按钮
- 答题反馈提示
- 统计题目进度

## 💡 游戏技巧

1. **难度选择**：
   - 新手建议从简单难度开始
   - 困难难度敌人强大但奖励更丰厚

2. **防御策略**：
   - 防御力越高，受到的伤害越少
   - 升级时优先提升防御力可以延长生存时间

3. **宝箱奖励**：
   - 集中答对题目可以多获宝箱
   - 选择适合自己的难度以确保答对率

4. **成就解锁**：
   - 记住特定等级的成就目标
   - 完美战士成就需要整场战斗不失血
