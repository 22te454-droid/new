# 📚 题库优化说明文档

## 概述
日语勇士养成游戏题库已全面优化和扩展，现在提供**三语支持**（日语为主，中文和英文为辅助），并包含更多分类合理、难度均衡的题目。

---

## 🎯 主要改进

### 1. **三语支持系统**
- **主语言**：日语（hiragana、katakana、kanji）
- **辅助语言**：中文（简体中文）和英文
- 每道题目都包含完整的多语翻译

**示例题目结构：**
```typescript
{
  id: 1,
  type: 'multiple',
  japanese: 'おはようございます',
  romanji: 'ohayou gozaimasu',
  meaning: {
    zh: '早上好（敬语）',
    en: 'Good morning (polite)'
  },
  options: ['おはようございます', 'こんにちは', 'こんばんは', 'おやすみなさい'],
  correct: 0
}
```

### 2. **题库规模和结构**

#### **简单难度 (初級)** - 30题
适合：日语零基础初学者，学习1-2周
主要内容：
- ✅ 基本问候（おはようございます、こんばんは等）
- ✅ 日常用语（感谢、道歉、肯定/否定）
- ✅ 基本名词（食物、动物、颜色、数字）
- ✅ 简单助词和基本动词
- ✅ 简单短句排序

**示例难度**：
- "ありがとう" = 感谢
- "猫" = 猫
- "私は学生＿です。" 填入正确助词

---

#### **中等难度 (中級)** - 20题  
适合：学过1-3个月的学习者
主要内容：
- ✅ 过去式和进行式（～ました、～ています）
- ✅ 希望表达（～たい、～たいです）
- ✅ 比较表达（より、ほど）
- ✅ 原因和结果（～ので、～から）
- ✅ 条件句（～たら、～ば、～なら）
- ✅ 能力表达（～ができます）
- ✅ 禁止和义务（～ないでください、～なければならない）
- ✅ 复杂短句组合

**示例难度**：
- "昨日、映画を見ました。" = 他/她昨天看了电影
- "私は日本に行きたいです。" = 我想去日本
- "AはBより大きいです。" = A比B大

---

#### **困难难度 (上級)** - 20题
适合：学过6个月以上的中高级学习者  
主要内容：
- ✅ 敬语系统（尊敬语、谦逊语）
- ✅ 复杂语法（～ながら、～てしまう、～ことができる）
- ✅ 假设和条件（～もし～たら、～べき）
- ✅ 被动和使役（受身、使役形）
- ✅ 高级连接词（～にもかかわらず、～によって）
- ✅ 高级词汇（革新、多角的、論理的等）

**示例难度**：
- "いらっしゃる" = 来/去/在（尊敬语）
- "音楽を聞きながら勉強する" = 一边听音乐一边学习
- "もし雨が降ったら、試合は中止になります。" = 如果下雨，比赛将取消

---

### 3. **学习模式支持**

通过 `learningMode` 变量，支持三种学习方式：
- `'jp'`：纯日语模式（显示日语和罗马字）
- `'cn'`：中文模式（显示中文含义）
- `'both'`：双语模式（显示日语和中文含义）✅ **推荐**

## 🎓 题目类型详解

### 1️⃣ **多选题 (Multiple Choice)**
- 展示日语、罗马字和中英文含义
- 需要选择正确的四个选项之一
- 涵盖词汇、语法、助词、动词形式等

```typescript
{
  type: 'multiple',
  japanese: '昨日、映画を見ました。',
  meaning: {
    zh: '昨天看了电影。',
    en: 'I watched a movie yesterday.'
  },
  options: ['見ます', '見ました', '見る', '見ている'],
  correct: 1  // 答案是第2个选项
}
```

### 2️⃣ **排序题 (Phrase Ordering)**
- 需要将乱序的日语短语重新排列成正确的顺序
- 展示中文含义和英文含义
- 帮助理解日语句子结构

```typescript
{
  type: 'phrase_order',
  meaning: {
    zh: '我昨天和朋友看了电影。',
    en: 'I watched a movie with my friend yesterday.'
  },
  phrase: ['私は', '昨日', '友達と', '映画を', '見ました'],
  correctOrder: [0, 1, 2, 3, 4]
}
```

## 🎯 使用建议

### 对初学者
1. 从**简单难度**开始
2. 建议使用 **'both'（双语）** 学习模式
3. 关注基本助词和常用表达

### 对中级学习者  
1. 适度挑战**中等难度**
2. 可切换到 **'jp'（纯日语）** 模式
3. 重点学习动词变位和语法结构

### 对高级学习者
1. 接受**困难难度**的挑战
2. 推荐使用 **'jp'（纯日语）** 模式
3. 掌握敬语系统和复杂语法
